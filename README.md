# chatbot-widget

Simply npm install
Make your changes and npm run build ;)

Add it to your website
```
<script src="./dist/widget.bundle.js" ></script>

<chat-widget 
    img-url="{path_to_logo}" 
    background-color="#fff" 
    popup-delay="300"
    sound-url="{optional_mp3_url}"
    >
</chat-widget>

```
